# SMG2-Restored-SMR.szs (v3.0)
This mod restores music and sounds from SMG in SMG2. For restored sounds, please check v2.0, v1.5 or v1.0 below. The latest update, v3.0, purely focuses on restored music. As of v3.0, [GLE-V4 beta](https://github.com/SuperHackio/GalaxyLevelEngine) is required!
